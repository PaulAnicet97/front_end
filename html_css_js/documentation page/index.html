Introduction

La cuisine ivoirienne comporte différents plats provenant de tous les groupes ethniques qui constituent la population du pays. Elle est en général très pimentée. La contribution de l'immigration joue également un rôle dans la conception de la cuisine ivoirienne

Sokossoko
